# ARC — Artificial Reasoning Corporation

> Bounded Search, Not Brute Force

Static landing site for [arc.ceo](https://arc.ceo).

**Stack**: Pure HTML/CSS/JS · GitHub Pages · GitHub Actions CI/CD

## Structure

```
index.html          Main site
assets/             SVG mark assets + founder photo
tasks/              TaskGuard project management
.github/workflows/  CI/CD pipeline
```

## Development

Open `index.html` in a browser. No build step required.

## Deployment

Push to `main` → GitHub Actions deploys to GitHub Pages → live at arc.ceo.
